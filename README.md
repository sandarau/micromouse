# micromouse

Designing and building an autonomous robot that can navigate from a corner of a 16 x 16 square maze to its central desitination square in the shortest possible time.

<u>Components used:</u>

- stm32f103c8t6 medium density
  
- ln298 dual motor driver (for testing stages)
  
- 5v buck regulator
  
- 2 dc motors
  
- 9v battery
  

Programming language used: **Rust**
